# Landing - Verifica

Esta carpeta contiene una landing page estática inspirada en la información pública de https://www.verifica.com.ar/.

Características implementadas:
- Diseño moderno y elegante con tipografía Inter
- Efecto parallax en el hero
- Animaciones: reveal on scroll, counters y tilt en tarjetas
- Secciones: Cómo funciona, Qué incluye, Tecnología, Testimonios y Contacto
- Responsive: adaptada a dispositivos móviles

Cómo probar localmente (Windows + XAMPP):
1. Copia la carpeta `landing` dentro de tu raíz del servidor local (por ejemplo `C:\xampp\htdocs\akina`).
2. Abre el archivo `index.html` directamente en un navegador o arranca Apache y visita `http://localhost/akina/landing/index.html`.

Notas legales:
- El contenido textual de la página está inspirado en la información pública del sitio oficial, y fue parafraseado y organizado para la demo.
- Imágenes y recursos son marcadores de posición (SVG). Reemplácelos por tu material si lo necesitas.

Si querés, puedo:
- Adaptar la paleta de colores, tipografías y el contenido textual exactamente como en la web original (si tenés permiso). 
- Agregar un formulario servidor/endpoint para envíos reales.
- Exportar como proyecto para Netlify o un contenedor Docker.
